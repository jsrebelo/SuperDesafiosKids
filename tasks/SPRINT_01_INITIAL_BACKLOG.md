# Sprint 01 — Fundação técnica do MVP

## Objetivo

Criar a aplicação base e os primeiros contratos do domínio.

## Tarefas

- TASK-001 Inicializar React, TypeScript e Vite.
- TASK-002 Configurar lint, format e typecheck.
- TASK-003 Criar estrutura de pastas.
- TASK-004 Definir entidade `ChildProfile`.
- TASK-005 Definir interface `ProfileRepository`.
- TASK-006 Criar repositório em memória.
- TASK-007 Implementar caso de utilização `CreateChildProfile`.
- TASK-008 Criar ecrã de seleção de perfil.
- TASK-009 Criar persistência local inicial.
- TASK-010 Adicionar testes unitários e integração.
- TASK-011 Configurar PWA.
- TASK-012 Validar funcionamento offline.

## Critério de sucesso

É possível criar e selecionar um perfil, fechar a aplicação, abrir novamente e recuperar o perfil sem Internet.
