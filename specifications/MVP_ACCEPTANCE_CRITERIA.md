# MVP_ACCEPTANCE_CRITERIA

## Perfis

- [ ] Criar e selecionar perfil.
- [ ] Recuperar perfil após reinício.
- [ ] Suportar vários perfis.

## Aprendizagem

- [ ] Dificuldade inicial depende da idade.
- [ ] Desempenho altera a dificuldade.
- [ ] Jogos não decidem dificuldade.

## Jogos

- [ ] Matemática funcional.
- [ ] Palavras funcional.
- [ ] Feedback positivo.
- [ ] Tentativas persistidas.

## Recompensas

- [ ] XP, moedas e estrelas funcionam.
- [ ] Não existem perdas.
- [ ] Desbloqueios persistem.

## Pais

- [ ] Área protegida.
- [ ] Estatísticas corretas.
- [ ] Limite diário.
- [ ] Definições persistem.

## Qualidade

- [ ] Funciona offline.
- [ ] PWA instalável.
- [ ] TypeScript strict.
- [ ] Testes aprovados.
- [ ] Acessibilidade validada.
- [ ] Sem publicidade.
- [ ] Sem recolha desnecessária.
- [ ] pt-PT consistente.
