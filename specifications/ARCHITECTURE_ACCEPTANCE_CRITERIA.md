# ARCHITECTURE_ACCEPTANCE_CRITERIA

- [ ] TypeScript usa `strict`.
- [ ] O domínio não importa React, Phaser ou APIs do navegador.
- [ ] Todos os jogos obtêm dificuldade através do ALE.
- [ ] Existe repositório em memória para testes.
- [ ] A aplicação inicia e funciona sem rede.
- [ ] Tentativas são persistidas atomicamente.
- [ ] Textos da interface usam internacionalização.
- [ ] Erros técnicos não expõem informação sensível.
- [ ] As regras pedagógicas têm testes unitários.
- [ ] A área infantil não contém ligações externas.
