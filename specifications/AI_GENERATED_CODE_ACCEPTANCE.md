# AI_GENERATED_CODE_ACCEPTANCE

Código criado ou alterado por IA só pode ser integrado quando:

- [ ] existe tarefa ou requisito relacionado;
- [ ] o plano foi coerente com a arquitetura;
- [ ] não foram inventadas funcionalidades;
- [ ] dependências novas foram aprovadas;
- [ ] TypeScript strict está aprovado;
- [ ] testes relevantes foram criados;
- [ ] lint, typecheck e testes passam;
- [ ] acessibilidade foi verificada;
- [ ] funcionamento offline foi verificado;
- [ ] não existem dados pessoais em logs;
- [ ] documentação foi atualizada;
- [ ] revisão humana foi concluída.
