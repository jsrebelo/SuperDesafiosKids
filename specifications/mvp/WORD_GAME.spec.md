# WORD_GAME.spec

## Objetivo

Completar palavras escolhendo a letra em falta.

## Dados necessários

- palavra;
- sílabas;
- imagem;
- idade mínima;
- dificuldade;
- letra removida;
- opções;
- pronúncia opcional.

## Regras

- usar português de Portugal;
- palavras concretas no MVP;
- letras em maiúsculas para níveis iniciais;
- esconder apenas uma letra;
- evitar palavras ambíguas;
- opções devem incluir a resposta correta;
- opções incorretas não podem formar outra resposta válida;
- respeitar acentos e cedilha.

## Dificuldade

- palavras de 3 a 5 letras para níveis iniciais;
- palavras maiores para níveis superiores;
- vogais primeiro;
- consoantes depois;
- posições internas antes de letras menos previsíveis.

## Critérios de aceitação

- [ ] Palavra corresponde à imagem.
- [ ] Resposta correta está disponível.
- [ ] Ortografia pt-PT é válida.
- [ ] Pronúncia pode ser repetida.
- [ ] Tentativa é persistida.
- [ ] Dificuldade vem do ALE.
