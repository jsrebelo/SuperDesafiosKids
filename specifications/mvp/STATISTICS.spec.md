# STATISTICS.spec

## Objetivo

Apresentar progresso de forma simples aos pais.

## Métricas

- sessões;
- duração;
- tentativas;
- respostas corretas;
- taxa de acerto;
- competências praticadas;
- nível estimado;
- sequência de utilização opcional.

## Regras

- não comparar crianças;
- não usar linguagem negativa;
- mostrar tendência, não diagnóstico;
- permitir filtro por período;
- estatísticas são calculadas localmente.

## Critérios de aceitação

- [ ] Métricas correspondem às tentativas.
- [ ] Perfis são isolados.
- [ ] Dados persistem.
- [ ] Área infantil não mostra relatórios complexos.
- [ ] Não são feitas inferências médicas ou psicológicas.
