# REWARD_SYSTEM.spec

## Objetivo

Motivar sem substituir o objetivo educativo.

## Recompensas do MVP

- XP;
- moedas;
- estrelas;
- autocolantes;
- avatares.

## Regras base

### Resposta correta

- XP base;
- moedas base;
- progresso para estrela.

### Tentativa com ajuda

- mantém feedback positivo;
- pode reduzir recompensa;
- nunca remove recompensas existentes.

### Resposta incorreta

- não perde XP;
- não perde moedas;
- recebe oportunidade de tentar novamente.

## Desbloqueios

- definidos por configuração;
- cosméticos;
- não bloqueiam conteúdo educativo.

## Critérios de aceitação

- [ ] Recompensa é determinística.
- [ ] Não existem recompensas negativas.
- [ ] Carteira persiste.
- [ ] Desbloqueios não se repetem.
- [ ] Sistema pode ser recalculado a partir de eventos.
