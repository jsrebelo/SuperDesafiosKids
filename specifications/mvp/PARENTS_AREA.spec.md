# PARENTS_AREA.spec

## Objetivo

Permitir gestão e acompanhamento por adultos.

## Funcionalidades

- gerir perfis;
- consultar estatísticas;
- definir limite diário;
- configurar áudio;
- ativar leitura por voz;
- ativar alto contraste;
- ativar movimento reduzido;
- exportação futura.

## Acesso

O MVP deve usar um desafio para adultos ou PIN configurável.

## Regras

- ações destrutivas exigem confirmação;
- limite diário não deve apagar progresso;
- ao atingir limite, mostrar mensagem calma;
- configurações são persistidas;
- não mostrar publicidade nem ofertas.

## Critérios de aceitação

- [ ] Área não abre acidentalmente.
- [ ] Perfis podem ser geridos.
- [ ] Estatísticas são corretas.
- [ ] Limite diário funciona.
- [ ] Definições persistem.
