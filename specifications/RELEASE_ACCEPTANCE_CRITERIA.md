# RELEASE_ACCEPTANCE_CRITERIA

- [ ] Não existem bugs Blocker ou Critical.
- [ ] Testes automatizados passam.
- [ ] Fluxos principais foram testados manualmente.
- [ ] Funciona sem rede.
- [ ] Progresso persiste após reinício.
- [ ] Migrações foram testadas.
- [ ] Acessibilidade foi validada.
- [ ] Não existem links externos na área infantil.
- [ ] Não existem dados pessoais em logs.
- [ ] Dependências foram auditadas.
- [ ] Changelog e versão foram atualizados.
- [ ] Existe plano de rollback.
- [ ] Release Test Report foi aprovado.
