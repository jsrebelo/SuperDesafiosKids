# BUTTON

## Variantes

- Primary
- Secondary
- Tertiary
- Icon
- Destructive (apenas área dos pais)

## Estados

- default
- hover
- focus
- pressed
- disabled
- loading

## Regras

- Área de toque ampla.
- Texto curto.
- Ícone opcional.
- Foco visível.
- Não usar variante destrutiva na área infantil.
