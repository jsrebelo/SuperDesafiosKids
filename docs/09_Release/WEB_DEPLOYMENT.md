# WEB_DEPLOYMENT

## Objetivo

Publicar a PWA.

## Requisitos

- HTTPS;
- service worker;
- manifest;
- ícones;
- página offline;
- cache controlado;
- headers de segurança;
- política de privacidade;
- testes em navegadores suportados.

## Rollout

- ambiente de preview;
- staging;
- produção;
- validação pós-publicação.
