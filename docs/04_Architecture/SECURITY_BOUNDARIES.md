# SECURITY_BOUNDARIES

## Área infantil

- sem links externos;
- sem compras;
- sem chat;
- sem recolha de texto livre;
- sem acesso às definições avançadas.

## Área dos pais

Protegida por desafio apropriado para adultos ou método configurável.

## Dados

- mínimos necessários;
- armazenamento local por defeito;
- exportação apenas por ação do adulto;
- sincronização futura requer consentimento;
- nenhuma publicidade comportamental.

## Dependências

Todas as dependências devem ser avaliadas quanto a:

- manutenção;
- vulnerabilidades;
- recolha de dados;
- tamanho;
- licença.
