# PARENTAL_CONSENT

## MVP offline

Não requer conta cloud.

## Funcionalidades futuras que exigem consentimento

- sincronização;
- exportação externa;
- telemetria;
- conta;
- notificações remotas;
- integração escolar.

## Regras

- consentimento explícito;
- linguagem clara;
- possibilidade de retirar consentimento;
- explicar finalidade;
- explicar retenção;
- não agrupar consentimentos independentes.
