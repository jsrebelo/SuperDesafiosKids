# DATA_RETENTION

## Local

Os dados permanecem no dispositivo até serem eliminados pelo adulto.

## Exportação

Só ocorre por ação explícita.

## Cloud futura

Deve definir:

- prazo de retenção;
- eliminação;
- portabilidade;
- revogação;
- cópias de segurança;
- localização dos dados.

## Regra

Nenhum dado deve ser mantido sem finalidade documentada.
