# SECURITY_AND_PRIVACY

## Princípios

- minimização de dados;
- armazenamento local por defeito;
- consentimento do adulto;
- sem publicidade comportamental;
- sem chat;
- sem recolha de texto livre infantil;
- sem links externos na área infantil;
- transparência;
- eliminação de dados controlada pelo adulto.

## Dados permitidos no MVP

- nome curto ou alcunha;
- faixa etária;
- avatar;
- progresso;
- preferências;
- estatísticas locais.

## Dados evitados

- data de nascimento completa;
- morada;
- localização;
- escola;
- fotografias;
- voz;
- contactos;
- identificadores publicitários.
