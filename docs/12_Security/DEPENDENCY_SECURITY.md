# DEPENDENCY_SECURITY

## Antes de adicionar uma dependência

Verificar:

- manutenção;
- licença;
- vulnerabilidades;
- tamanho;
- recolha de dados;
- compatibilidade;
- necessidade real.

## Processo

- atualizações periódicas;
- auditoria automática;
- revisão manual de vulnerabilidades críticas;
- remoção de bibliotecas abandonadas;
- lockfile obrigatório.
