# THREAT_MODEL

## Ameaças principais

- acesso indevido à área dos pais;
- perda de progresso;
- dependência maliciosa;
- fuga de dados;
- manipulação de recompensas;
- ficheiros de assets adulterados;
- sincronização futura insegura;
- links externos acidentais.

## Mitigações

- separação de áreas;
- validação;
- armazenamento local;
- atualizações controladas;
- revisão de dependências;
- ausência de dados desnecessários;
- logs sem informação pessoal.
