# STORAGE_SECURITY

## Regras

- usar APIs seguras do dispositivo;
- evitar dados sensíveis em texto simples quando existir alternativa;
- proteger exportações;
- validar integridade;
- suportar migrações;
- não apagar progresso silenciosamente;
- separar área infantil de configurações adultas.

## Web

Usar armazenamento local isolado por origem.

## Mobile

Usar armazenamento da aplicação e mecanismos seguros do sistema quando necessário.
