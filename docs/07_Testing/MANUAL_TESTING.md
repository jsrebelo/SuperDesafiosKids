# MANUAL_TESTING

## Áreas

- usabilidade infantil;
- clareza das instruções;
- resposta ao toque;
- áudio;
- animações;
- acessibilidade;
- desempenho;
- offline;
- recuperação de erros.

## Sessões com crianças

Quando existirem testes de utilização reais:

- exigir consentimento do responsável;
- recolher o mínimo de dados;
- não gravar áudio ou vídeo sem autorização;
- evitar perguntas invasivas;
- permitir interrupção imediata;
- documentar apenas observações necessárias.
