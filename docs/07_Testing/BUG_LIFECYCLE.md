# BUG_LIFECYCLE

## Estados

- New
- Triaged
- Ready
- In Progress
- In Review
- Verified
- Closed
- Reopened

## Severidade

### Blocker

Impede utilização principal ou causa perda de dados.

### Critical

Afeta segurança, privacidade ou progresso.

### Major

Afeta funcionalidade importante sem solução aceitável.

### Minor

Problema limitado com alternativa disponível.

### Cosmetic

Problema visual sem impacto funcional.

## Regra

Bugs de privacidade, perda de dados ou segurança bloqueiam release.
