# E2E_TESTING

## Ferramenta

Playwright.

## Fluxos obrigatórios

1. Criar perfil.
2. Selecionar perfil.
3. Jogar matemática.
4. Submeter resposta.
5. Receber feedback.
6. Fechar e reabrir.
7. Confirmar progresso.
8. Abrir área dos pais.
9. Consultar estatísticas.
10. Ativar modo offline.
11. Repetir sessão sem rede.

## Regra

Os testes end-to-end devem usar dados isolados e poder correr em CI.
