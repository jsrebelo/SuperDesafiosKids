# UI_AND_ACCESSIBILITY_TESTING

## Ferramentas

- React Testing Library
- axe-core ou equivalente
- Playwright

## Verificações

- foco visível;
- ordem de tabulação;
- rótulos;
- contraste;
- tamanho dos alvos;
- movimento reduzido;
- ausência de dependência de hover;
- leitores de ecrã;
- mensagens positivas;
- suporte a zoom;
- estados sem depender apenas de cor.
