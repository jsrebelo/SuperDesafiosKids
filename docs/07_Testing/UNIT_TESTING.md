# UNIT_TESTING

## Ferramenta

Vitest.

## Regras

- testes determinísticos;
- controlar aleatoriedade;
- controlar relógio;
- usar builders ou factories;
- testar regras, não implementação interna;
- evitar mocks excessivos;
- usar repositórios em memória.

## Cobertura prioritária

- dificuldade;
- domínio de competência;
- repetição;
- cálculo de recompensas;
- validação de respostas;
- limites por idade.
