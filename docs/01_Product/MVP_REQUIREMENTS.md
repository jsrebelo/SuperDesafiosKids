# MVP_REQUIREMENTS

## Requisitos funcionais

### Perfis

- `FR-MVP-001` Criar perfil infantil.
- `FR-MVP-002` Editar nome, idade e avatar.
- `FR-MVP-003` Selecionar perfil ativo.
- `FR-MVP-004` Arquivar perfil mediante confirmação parental.
- `FR-MVP-005` Suportar vários perfis no mesmo dispositivo.

### Matemática

- `FR-MVP-010` Gerar exercícios de contagem.
- `FR-MVP-011` Gerar somas.
- `FR-MVP-012` Gerar subtrações.
- `FR-MVP-013` Gerar multiplicações para níveis adequados.
- `FR-MVP-014` Gerar divisões simples para níveis adequados.
- `FR-MVP-015` Validar respostas.
- `FR-MVP-016` Adaptar dificuldade.

### Palavras

- `FR-MVP-020` Selecionar palavra adequada à idade.
- `FR-MVP-021` Esconder uma letra.
- `FR-MVP-022` Gerar opções de resposta.
- `FR-MVP-023` Validar a letra escolhida.
- `FR-MVP-024` Reproduzir pronúncia quando disponível.

### Recompensas

- `FR-MVP-030` Atribuir XP.
- `FR-MVP-031` Atribuir moedas.
- `FR-MVP-032` Atribuir estrelas.
- `FR-MVP-033` Desbloquear recompensas cosméticas.

### Estatísticas

- `FR-MVP-040` Guardar tentativas.
- `FR-MVP-041` Guardar duração de sessões.
- `FR-MVP-042` Calcular taxa de acerto.
- `FR-MVP-043` Mostrar progresso por competência.

### Área dos pais

- `FR-MVP-050` Proteger o acesso.
- `FR-MVP-051` Consultar perfis.
- `FR-MVP-052` Consultar estatísticas.
- `FR-MVP-053` Definir limite diário.
- `FR-MVP-054` Configurar áudio e acessibilidade.

## Requisitos não funcionais

- `NFR-MVP-001` Funcionar sem Internet.
- `NFR-MVP-002` Usar TypeScript strict.
- `NFR-MVP-003` Persistir progresso após cada tentativa.
- `NFR-MVP-004` Iniciar em até 3 segundos no dispositivo alvo.
- `NFR-MVP-005` Suportar toque e teclado.
- `NFR-MVP-006` Não depender apenas da cor.
- `NFR-MVP-007` Não recolher dados desnecessários.
- `NFR-MVP-008` Não apresentar publicidade.
- `NFR-MVP-009` Usar pt-PT.
- `NFR-MVP-010` Ter testes automatizados para regras centrais.
