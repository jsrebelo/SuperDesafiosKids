# ACCESSIBILITY_VISUAL

## Requisitos

- contraste adequado;
- foco visível;
- navegação por teclado na Web;
- leitores de ecrã em áreas não baseadas em canvas;
- texto alternativo para imagens informativas;
- opção de movimento reduzido;
- opção de alto contraste;
- opção de fonte amigável para dislexia;
- não usar cor como único indicador.

## Jogos em canvas

Quando Phaser for usado:

- fornecer descrição textual equivalente quando possível;
- expor ações principais fora do canvas;
- manter botões essenciais em elementos HTML;
- garantir alternativa para instruções importantes.
