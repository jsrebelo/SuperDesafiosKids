# UI_UX_PRINCIPLES

**Versão:** 0.6.0

## Objetivo

Garantir que a interface é clara, segura, divertida e utilizável por crianças dos 5 aos 10 anos.

## Princípios

1. Um objetivo principal por ecrã.
2. Botões grandes e claramente identificáveis.
3. Pouco texto por ecrã.
4. Ícones acompanhados por texto quando necessário.
5. Feedback imediato após cada ação.
6. Nunca usar medo, pressão ou penalização.
7. Evitar menus profundos.
8. Manter padrões de interação consistentes.
9. Permitir utilização por toque.
10. Diferenciar visualmente a área infantil da área dos pais.

## Regras

- O ecrã principal deve apresentar no máximo seis ações principais.
- Um botão primário deve ser visualmente dominante.
- Elementos interativos devem ter área mínima confortável para toque.
- A criança deve conseguir regressar ao início em até duas ações.
- Toda a interação importante deve ter confirmação visual.
