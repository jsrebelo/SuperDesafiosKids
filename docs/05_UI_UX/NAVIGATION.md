# NAVIGATION

## Estrutura principal

```text
Seleção de Perfil
→ Início
→ Escolha de Jogo
→ Sessão
→ Recompensa
→ Próximo Desafio ou Início
```

## Área dos pais

```text
Início
→ Acesso dos Pais
→ Painel
→ Perfis / Limites / Estatísticas / Definições
```

## Regras

- A navegação infantil utiliza no máximo dois níveis.
- O botão “Voltar” deve manter posição consistente.
- Sair de uma sessão deve pedir confirmação simples.
- A área dos pais nunca deve ser aberta acidentalmente.
- O perfil ativo deve estar sempre visível no ecrã inicial.
