# ANIMATION_GUIDELINES

## Objetivo

Usar movimento para orientar, celebrar e explicar.

## Duração recomendada

- microinterações: 100–200 ms;
- transições: 200–400 ms;
- celebrações: até 1200 ms;
- animações educativas: apenas enquanto forem úteis.

## Regras

- Não bloquear a criança com animações longas.
- Respeitar preferência de movimento reduzido.
- Evitar flashes rápidos.
- Não usar movimento contínuo em vários elementos ao mesmo tempo.
- A celebração deve poder ser ignorada após o primeiro instante.
