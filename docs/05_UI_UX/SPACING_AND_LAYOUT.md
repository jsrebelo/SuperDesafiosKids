# SPACING_AND_LAYOUT

## Escala base

Usar múltiplos consistentes:

- 4
- 8
- 12
- 16
- 24
- 32
- 48

## Regras

- Elementos interativos não devem ficar demasiado próximos.
- Cartões devem ter espaço interno generoso.
- O conteúdo principal deve ficar dentro de uma área segura.
- A interface deve evitar densidade excessiva.
- O layout deve adaptar-se primeiro a tablet e telemóvel.
