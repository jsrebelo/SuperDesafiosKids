# TYPOGRAPHY

## Objetivos

- Leitura simples.
- Formas de letras claras.
- Boa distinção entre caracteres semelhantes.
- Tamanhos adequados a tablets e telemóveis.

## Escala semântica

- `display`
- `heading.1`
- `heading.2`
- `heading.3`
- `body.large`
- `body`
- `label`
- `caption`

## Regras

- Evitar blocos longos de texto.
- Usar alinhamento à esquerda para leitura.
- Nunca usar apenas maiúsculas em frases longas.
- Permitir opção de fonte amigável para dislexia.
- O tamanho mínimo de texto infantil deve ser superior ao usado em interfaces adultas comuns.
