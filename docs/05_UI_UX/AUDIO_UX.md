# AUDIO_UX

## Regras

- Todo o som deve poder ser desligado.
- Música e efeitos devem ter controlos separados.
- Feedback sonoro nunca substitui feedback visual.
- Áudio não deve assustar ou ser excessivamente alto.
- Instruções faladas devem poder ser repetidas.
- A aplicação deve respeitar definições do dispositivo.
