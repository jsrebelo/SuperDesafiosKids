# RESPONSIVE_DESIGN

## Dispositivos alvo

- Telemóvel vertical.
- Telemóvel horizontal.
- Tablet.
- Desktop.

## Regras

- A experiência infantil deve funcionar sem depender de hover.
- Em ecrãs pequenos, apresentar uma coluna.
- Em tablets, usar duas colunas quando melhorar a compreensão.
- Jogos com duas imagens devem adaptar zoom e disposição.
- Elementos importantes devem permanecer na área segura.
- Não esconder ações essenciais em menus contextuais.
