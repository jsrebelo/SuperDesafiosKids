# FEEDBACK_STATES

## Resposta correta

- animação breve;
- som opcional;
- mensagem positiva;
- recompensa visível;
- transição automática ou botão “Continuar”.

## Resposta incorreta

Nunca mostrar “Errado”.

Usar:

- “Boa tentativa!”
- “Vamos pensar juntos.”
- dica opcional;
- nova tentativa;
- simplificação do desafio quando necessário.

## Estados técnicos

- carregamento;
- vazio;
- sem ligação;
- erro recuperável;
- erro de armazenamento.

A criança deve receber linguagem simples. O adulto pode receber detalhes adicionais.
