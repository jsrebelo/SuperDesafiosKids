# OFFLINE_AND_DATA_RULES

## Offline

- funcionalidades essenciais não dependem de rede;
- assets do MVP devem estar disponíveis localmente;
- falhas de rede não bloqueiam a criança;
- sincronização é futura e opcional.

## Dados

- recolher apenas o necessário;
- não guardar data de nascimento completa;
- não enviar dados infantis sem consentimento;
- não incluir nomes em logs;
- persistir tentativas de forma atómica;
- migrações não podem apagar progresso.
