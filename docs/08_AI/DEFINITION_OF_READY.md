# DEFINITION_OF_READY

Uma tarefa está pronta quando:

- [ ] tem objetivo claro;
- [ ] identifica requisitos relacionados;
- [ ] possui critérios de aceitação;
- [ ] dependências estão identificadas;
- [ ] desenhos ou fluxos necessários existem;
- [ ] regras pedagógicas estão definidas;
- [ ] dúvidas importantes foram resolvidas;
- [ ] pode ser testada.
