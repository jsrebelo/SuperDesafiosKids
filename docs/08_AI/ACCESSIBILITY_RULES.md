# ACCESSIBILITY_RULES

Toda a funcionalidade deve verificar:

- foco visível;
- ordem de tabulação;
- rótulos acessíveis;
- contraste;
- texto alternativo;
- movimento reduzido;
- interação sem hover;
- estado não comunicado apenas por cor;
- alternativa HTML para ações essenciais de canvas.

A acessibilidade faz parte da Definition of Done.
