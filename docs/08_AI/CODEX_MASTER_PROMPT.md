## Regras obrigatórias

- React + TypeScript strict + Vite.
- Domínio independente de frameworks.
- Offline First.
- UI não acede diretamente ao armazenamento.
- Jogos usam o Adaptive Learning Engine.
- Regras pedagógicas ficam no domínio.
- Textos em pt-PT através de internacionalização.
- Sem publicidade.
- Sem compras.
- Sem dados desnecessários.
- Testes obrigatórios.
- Acessibilidade obrigatória.

## Método

1. Resume a tarefa.
2. Lista ficheiros a criar ou alterar.
3. Implementa o menor incremento funcional.
4. Adiciona testes.
5. Executa lint, typecheck e testes.
6. Verifica offline e acessibilidade.
7. Atualiza documentação.
8. Resume o resultado e limitações.

Não inventes requisitos nem alteres a arquitetura sem ADR.
