# EDUCATIONAL_OBJECTS

## Categorias iniciais

- animais;
- frutas;
- legumes;
- brinquedos;
- transportes;
- roupa;
- profissões;
- objetos da casa;
- escola;
- natureza;
- instrumentos;
- alimentos;
- formas;
- números;
- letras.

## Metadados obrigatórios

Cada objeto deve ter:

- `id`;
- nome em pt-PT;
- categoria;
- dificuldade linguística;
- faixa etária recomendada;
- descrição curta;
- texto alternativo;
- ficheiro de imagem;
- variantes linguísticas futuras.

## Regras

- preferir objetos reconhecíveis;
- evitar regionalismos pouco claros;
- validar ortografia em pt-PT;
- imagens não devem conter texto embutido.
