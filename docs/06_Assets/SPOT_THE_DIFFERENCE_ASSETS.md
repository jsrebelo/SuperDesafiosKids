# SPOT_THE_DIFFERENCE_ASSETS

## Estrutura de cada conjunto

- imagem base;
- imagem modificada;
- lista de diferenças;
- coordenadas das zonas;
- tolerância de toque;
- dificuldade;
- faixa etária;
- descrição acessível.

## Tipos de diferença

- objeto removido;
- objeto adicionado;
- mudança de cor;
- mudança de posição;
- mudança de tamanho;
- detalhe alterado.

## Regras

- diferenças devem ser objetivas;
- não usar alterações demasiado pequenas;
- evitar diferenças apenas por cor;
- não sobrepor zonas de toque;
- a dificuldade deve depender da idade;
- cada par deve ser testado em telemóvel e tablet.
