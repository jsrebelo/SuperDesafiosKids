# LICENSING_AND_RIGHTS

## Regras

- usar apenas assets originais, licenciados ou de domínio permitido;
- guardar prova de licença;
- não usar personagens protegidas;
- não usar marcas sem autorização;
- confirmar direitos de música e voz;
- contratos de produção devem permitir uso comercial;
- fontes devem ter licença compatível.

## Registo obrigatório

Cada asset externo deve indicar:

- autor;
- origem;
- licença;
- data de obtenção;
- restrições;
- prova documental.
