# ASSET_NAMING_CONVENTION

## Formato

```text
<categoria>_<tema>_<nome>_<variante>_<estado>.<ext>
```

## Exemplos

```text
character_forest_fox_idle.webp
character_forest_fox_happy.webp
background_space_level01.webp
object_food_apple_default.webp
icon_navigation_home.svg
audio_feedback_correct_01.ogg
```

## Regras

- minúsculas;
- inglês técnico nos nomes de ficheiro;
- sem espaços;
- sem acentos;
- usar underscore;
- incluir variante quando necessário;
- não incluir datas no nome.
