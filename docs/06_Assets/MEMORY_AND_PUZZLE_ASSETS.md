# MEMORY_AND_PUZZLE_ASSETS

## Memória

Cada par deve ter:

- imagem frontal;
- versão acessível;
- categoria;
- nível de dificuldade;
- associação clara.

## Puzzle

Cada imagem deve ter:

- versão completa;
- metadados de corte;
- número de peças recomendado;
- área segura;
- descrição textual.

## Regras

- imagens simples para idades mais baixas;
- aumentar detalhe gradualmente;
- evitar peças visualmente indistinguíveis;
- manter boa leitura das bordas.
