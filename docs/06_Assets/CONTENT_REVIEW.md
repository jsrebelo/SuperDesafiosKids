# CONTENT_REVIEW

## Revisão pedagógica

Verificar:

- adequação à idade;
- clareza;
- dificuldade;
- ortografia;
- coerência curricular;
- ausência de ambiguidades.

## Revisão visual

Verificar:

- estilo;
- contraste;
- legibilidade;
- tamanho;
- consistência;
- ausência de elementos impróprios.

## Revisão técnica

Verificar:

- formato;
- tamanho;
- nome;
- compressão;
- metadados;
- carregamento offline.
