# VOICE_AND_NARRATION

## Utilizações

- leitura de instruções;
- pronúncia de palavras;
- feedback;
- ajuda;
- nomes de objetos.

## Regras

- português de Portugal;
- dicção clara;
- ritmo lento, mas natural;
- tom positivo;
- frases curtas;
- áudio repetível;
- texto visível equivalente sempre disponível.
