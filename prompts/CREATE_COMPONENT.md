# Prompt — Criar componente

Cria o componente `[nome]` com base em `[especificação]`.

Requisitos:

- usar tokens do Design System;
- suportar teclado e leitor de ecrã;
- incluir estados necessários;
- não incluir regras de domínio;
- não usar valores visuais arbitrários;
- criar testes de interface;
- escrever Storybook apenas quando o projeto o adotar.
