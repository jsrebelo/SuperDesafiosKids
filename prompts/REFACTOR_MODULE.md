# Prompt — Refatorar módulo

Refatora `[módulo]` sem alterar comportamento observável.

Antes:

- identifica testes existentes;
- descreve problemas;
- define limites da refatoração.

Durante:

- preserva contratos públicos;
- reduz acoplamento;
- elimina duplicação útil;
- não adiciona dependências sem aprovação.

Depois:

- executa testes;
- compara comportamento;
- atualiza documentação quando necessário.
