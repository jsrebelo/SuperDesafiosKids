# Prompt — Rever código

Revê as alterações em `[branch/ficheiros]`.

Verifica:

- requisitos;
- arquitetura;
- regras pedagógicas;
- TypeScript;
- testes;
- acessibilidade;
- offline;
- privacidade;
- internacionalização;
- desempenho.

Classifica problemas como:

- bloqueador;
- importante;
- melhoria.

Não alterar código durante a primeira revisão.
