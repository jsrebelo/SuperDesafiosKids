# Prompt — Escrever testes

Cria testes para `[módulo/funcionalidade]`.

Cobrir:

- comportamento principal;
- casos limite;
- erros esperados;
- acessibilidade quando aplicável;
- persistência e offline quando aplicável.

Não testar detalhes internos sem valor. Controlar aleatoriedade, relógio e armazenamento.
